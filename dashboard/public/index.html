claude

Gemini حالا مثل GapGPT با یک API Key کار می‌کند — نیازی به نصب CLI نیست. کلید را رایگان از Google AI Studio بگیرید و در تنظیمات وارد کنید.
① دریافت API Key
به Google AI Studio بروید و یک کلید API بسازید:
https://aistudio.google.com/apikey
② ذخیره‌ی کلید در تنظیمات
از داشبورد، چرخ‌دنده‌ی تنظیمات ← بخش Gemini، کلید را در فیلد «کلید API جمینای» وارد و ذخیره کنید. (در صورت تمایل می‌توانید مدل را هم در «مدل» مشخص کنید — پیش‌فرض gemini-2.5-pro.)
به‌جای داشبورد می‌توانید مستقیم در فایل secrets.env هم اضافه کنید:
GEMINI_API_KEY=AIza...your-key-here
③ اجرای پل
node claude-bridge.js
سپس از داشبورد، موتور را به «Gemini» تغییر دهید. تغییر کلید بدون ری‌استارت پل اعمال می‌شود.
Ollama مدل‌ها را روی همین دستگاه به‌صورت محلی اجرا می‌کند — برای مواقع اضطراری (بدون اینترنت / تمام‌شدن سهمیه). نیازی به API Key ندارد.
① نصب Ollama
brew install ollama
یا از https://ollama.com/download دانلود کنید.
نصب‌کننده را از سایت دانلود و اجرا کنید:
https://ollama.com/download/windows
curl -fsSL https://ollama.com/install.sh | sh
② دانلود یک مدل
ollama pull llama3.1
می‌توانید هر مدل دیگری را هم بکشید (مثلاً qwen2.5-coder) و نامش را در تنظیمات بخش Ollama وارد کنید.
③ اجرای سرویس Ollama
ollama serve
روی پورت 11434 گوش می‌دهد. (در ویندوز/مک معمولاً بعد از نصب خودکار اجرا می‌شود.)
④ تنظیمات و اجرای پل
در تنظیمات بخش Ollama، در صورت نیاز آدرس پایه و نام مدل را تغییر دهید (پیش‌فرض‌ها معمولاً کافی‌اند). سپس پل را اجرا کنید:
node claude-bridge.js
سپس از داشبورد، موتور را به «Ollama (محلی)» تغییر دهید.
GapGPT یک سرویس OpenAI-compatible ایران‌پسند است. نیاز به نصب CLI ندارد — فقط یک API Key لازم است.
① دریافت API Key
به سایت GapGPT بروید و یک کلید API بسازید:
https://gapgpt.app
② ذخیره‌ی کلید در فایل secrets.env
در پوشه‌ی پروژه، فایل secrets.env را باز کنید و این خط را اضافه کنید:
GAPGPT_API_KEY=sk-xxxx-your-key-here
یا با ترمینال:
echo 'GAPGPT_API_KEY=sk-xxxx' >> secrets.env
فایل secrets.env را در Notepad باز کنید و اضافه کنید:
GAPGPT_API_KEY=sk-xxxx-your-key-here
یا با PowerShell:
Add-Content secrets.env 'GAPGPT_API_KEY=sk-xxxx'
③ اجرای پل
node claude-bridge.js
سپس از داشبورد، موتور را به «GapGPT» تغییر دهید.
