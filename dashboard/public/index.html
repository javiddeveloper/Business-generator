claude

GapGPT یک سرویس OpenAI-compatible ایران‌پسند است. نیاز به نصب CLI ندارد — فقط یک API Key لازم است.
① دریافت API Key
به سایت GapGPT بروید و یک کلید API بسازید:
https://gapgpt.app
② ذخیره‌ی کلید در فایل secrets.env
در پوشه‌ی پروژه، فایل secrets.env را باز کنید و این خط را اضافه کنید:
GAPGPT_API_KEY=sk-xxxx-your-key-here
یا با ترمینال:
echo 'GAPGPT_API_KEY=sk-xxxx' >> secrets.env
فایل secrets.env را در Notepad باز کنید و اضافه کنید:
GAPGPT_API_KEY=sk-xxxx-your-key-here
یا با PowerShell:
Add-Content secrets.env 'GAPGPT_API_KEY=sk-xxxx'
③ اجرای پل
node claude-bridge.js
سپس از داشبورد، موتور را به «GapGPT» تغییر دهید.
